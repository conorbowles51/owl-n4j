## Typography

### Font Families

- **Body and interface** (`font-sans`): `Source Sans 3, "Segoe UI", sans-serif`
- **Headings**: `Manrope, "Segoe UI", sans-serif`
- **Monospace** (`font-mono`): `IBM Plex Mono, "SFMono-Regular", Consolas, monospace`

### Font Scale (Data-Dense)

Loupe uses a compact type scale optimized for information density:

| Class | Size | Line Height | Usage |
|-------|------|-------------|-------|
| `text-xs` | 12px | 16px | Metadata, timestamps, labels |
| `text-sm` | 13px | 20px | Body text, table cells |
| `text-base` | 14px | 22px | Default body, inputs |
| `text-lg` | 16px | 24px | Section headers |
| `text-xl` | 18px | 26px | Page titles |
| `text-2xl` | 22px | 28px | Dashboard headings |
| `text-3xl` | 28px | 34px | Hero text (rare) |

### Font Weight Rules

| Weight | Value | Usage |
|--------|-------|-------|
| Regular | 400 | Body text, descriptions |
| Medium | 500 | Table headers, labels, nav items |
| Semibold | 600 | Headings, card titles, buttons, badges |
| Bold | 700 | Rarely used — prefer semibold |

## Spacing

### Spacing Conventions

| Context | Spacing | Tailwind Class |
|---------|---------|---------------|
| Component internal padding | 12–16px | `p-3` / `p-4` |
| Card padding | 16px | `p-4` |
| Between cards/sections | 16px | `gap-4` |
| Between inline items | 6–8px | `gap-1.5` / `gap-2` |
| Page content padding | 24px | `p-6` |
| Dense table cell padding | 8–12px | `px-2 py-1.5` |
| Badge internal padding | 8px horiz | `px-2 py-0.5` |
| Button padding (default) | 14px horiz | `px-3.5` |

### Layout Dimensions

| Element | Size |
|---------|------|

| Sidebar expanded | 240px |
| Side panels (details, chat) | ~360px, resizable |

## Component Rules

### Buttons
- **Primary** (amber): Main actions — "Save", "Create", "Upload"
- **Secondary** (slate): Alternate actions — "Cancel" paired with Primary
- **Outline**: Tertiary actions, less visual weight
- **Ghost**: Toolbar actions, icon buttons, inline actions
- **Danger** (red): Destructive actions — "Delete", "Remove"
- **Link**: Text-style navigation buttons

### Focus States
All interactive elements use **amber focus ring**: `focus-visible:ring-2 focus-visible:ring-amber-500/40`

### Dark Mode
- Dark mode is the **default** — design for dark first
- Every component must look correct in dark mode
- Use semantic color tokens (`bg-card`, `text-foreground`, `border-border`), not raw colors
- Exception: node palette colors are fixed (not theme-dependent)

### Data Density
- Base font size is 14px (not the typical 16px)
- Use compact spacing — investigators need information density
- Truncate long text with `truncate` class + tooltip on hover
- Tables use dense cell padding (`px-2 py-1.5`)
