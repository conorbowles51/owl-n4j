08

Iconography

Use Lucide icons exclusively. 16px for inline/navigation, 20px for buttons and emphasis, 24px for empty states and headers. Stroke width 1.5px for consistency.

Library: Lucide React (lucide-react) — Open source, MIT licensed, consistent 24x24 grid, tree-shakeable. Do not mix with other icon libraries. Stroke width: 1.5px default, 2px for emphasis.
09

Brand Rules

Do
  • Use dark mode as the default experience
  • Prioritize data density — investigators need information, not whitespace
  • Use amber only for primary actions, focus states, and key highlights
  • Keep the UI quiet — let the data speak
  • Use the monospace font for all machine-generated or technical content
  • Maintain consistent spacing on the 4px grid
  • Use the graph node palette consistently across all views
  • Test all color combinations for WCAG AA contrast (4.5:1 minimum)
Don't
  • Use amber for large background areas — it's an accent, not a surface
  • Mix icon libraries — Lucide only
  • Add decorative animations — all motion must be functional
  • Use gradients on UI surfaces (reserved for marketing)
  • Put data in light-on-light combinations — ensure contrast
  • Use rounded-full on containers (only pills, avatars, badges)
  • Create new grays — use the slate scale exclusively
  • Override shadcn components without going through the theme layer
