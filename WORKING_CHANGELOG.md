# Working Changelog

## 2026-07-05

- Folder notes now get used when creating document and audio summaries, not just when finding people, places, and relationships.
- Users can now edit important entity details from the side panel, such as name, type, summary, notes, date, time, and location.
- User edits are protected so future AI processing should not quietly overwrite the details an investigator has corrected.
- The edit screen now hides confusing technical fields and focuses on the details investigators actually need to change.
- Timeline events can now have a specific time added, and events with times are ordered correctly within each day.
- Location editing is easier: users can search for an address, adjust the map pin, or clear the location.
- Technical source details are still available, but they are tucked away so they do not clutter the main details view.
- The edit window was redesigned to be wider, simpler, and easier to read, with a larger map and better-sized text boxes.
- The timeline is easier to read: each section is now a specific day, event times are clearer, timed events appear in order, and selected events no longer use clashing colors.
- A new Notebook panel is being added so investigators can write case notes from anywhere in the case.
- Notes can be linked to important people, places, events, and evidence files, so they are easier to find again later.
- Your own notebook notes now say "You", while the small initials badge still represents the note author.
- Investigators can now save a focused version of the timeline, so they can keep only the events that matter for a particular review.
- Timeline events can be selected in a new curation mode without losing the normal click-to-open details behavior.
- Timelines can now be exported as CSV or as a polished PDF with clear day sections, event times, summaries, linked people or places, and a source appendix.

## 2026-07-07

- Evidence processing is being tightened so dates, times, locations, and payment details are saved in the right place more reliably.
- Timeline items with a real time hidden inside uploaded phone or media data should now be less likely to show as "No time".
- Phone extraction data is being cleaned up more consistently before it reaches the timeline, map, and exports.
- Admins can now see when a platform update is available and start the update from the admin area, instead of logging into the server manually.
